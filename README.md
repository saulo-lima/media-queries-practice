#  Responsive Media Queries

Projeto desenvolvido com foco em **responsividade**, utilizando o conceito de **Mobile First** e **Media Queries** para adaptar o layout em diferentes dispositivos.

---

##  Tecnologias utilizadas

* HTML5
* CSS3

---

##  Objetivo

Praticar a adaptação de interfaces para múltiplos tamanhos de tela, incluindo:

* 📱 Smartphones
* 📲 Tablets
* 💻 Desktops
* 📺 TVs
* 🖨️ Impressão

---

##  Conceitos aplicados

* Mobile First
* Media Queries
* Breakpoints
* Responsividade
* Organização de CSS

---

## 📸 Comportamento do projeto

O layout se adapta automaticamente conforme o tamanho da tela, alterando:

* Imagens exibidas
* Plano de fundo
* Estrutura do conteúdo

---

## Aprendizados

Durante o desenvolvimento, foi possível entender melhor:

* Como funcionam os breakpoints
* Como aplicar Media Queries corretamente
* A importância de começar pelo mobile
